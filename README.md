# Notice Management Server

## Overview
Backend API for Notice Management Web App built using Vercel Serverless Functions and MongoDB Native Driver.

## Tech Stack
- Node.js
- Vercel Serverless Functions
- MongoDB (Native Driver)

## API Base URL
https://your-server.vercel.app/api

## Endpoints
- POST /api/notices
- GET /api/notices
- GET /api/notices/:id
- PATCH /api/notices/:id

## Installation
npm install

## Environment Variables
Create `.env` file:
MONGO_URI=your_mongodb_uri
MONGO_DB=notice_management

## Deployment
Deploy directly on Vercel.
